Convore.js by danielrmz

Convore for Node.js. Provides a set of wrappers around their HTTP API

Install

Simply: 

git clone git://github.com/danielrmz/Convore.js.git


var Convore = require("./Convore.js"); 
var client = new Convore("user","passwd");
client.Live(function(message) { /* callback */ });
